LEO_StarNet_EMC_V7.0 工程交付版 使用说明
==================================================

一、工程内容
1. 主入口文件
   - LEO_StarNet_EMC_V7_0_Engineering.m
2. 默认运行脚本
   - run_V7_default.m

二、建议环境
1. MATLAB R2021a
2. 建议工具箱（按工程中启用模块而定）：
   - Deep Learning Toolbox
   - Global Optimization Toolbox
   - Signal Processing Toolbox

三、快速开始
方式1：默认模式运行
    run('run_V7_default.m')

方式2：主入口运行
    LEO_StarNet_EMC_V7_0_Engineering

四、默认模式说明
1. 工程提供默认参数模式。
2. 当未手动输入配置参数时，程序将自动使用内置默认参数运行。
3. 默认关键要求参考值包括：
   - 下行频率：1.5 GHz
   - 上行频率：1.6 GHz
   - 语音业务速率：2.4 kbps（需求参考）
   - 宽带通信速率目标：> 20 Mbps
   - 接收灵敏度目标：优于 -120 dBm
   - 路测系统通信信噪比目标：>= 1 dB
   - 抗多普勒频率变化率目标：300 Hz/s

五、V7 主要改进点
1. 新增上行链路（Uplink）仿真能力
2. 双链路展示
3. 参数配置能力（未输入时走默认参数）
4. 合规性检查
5. 结果保存

六、参数配置建议
后续若对接单位提供以下参数，可进一步替换默认代理参数：
1. 接收机端口接收信号功率
2. 天线参数
3. 天线系数 AF
4. 转换系数
5. 线损、前放增益、罩损等修正项

七、运行建议
1. 首次运行建议先用默认模式，确认主流程跑通。
2. 再逐步开启：
   - 最劣工况搜索
   - 干扰识别
   - 图像导出

八、说明
1. 当前 V7 上行链路部分包含工程化代理建模，用于在现有下行代码框架上快速形成可用交付版本。
2. 当后续拿到更详细的端口级、天线级、实验室换算参数后，建议继续将代理模型替换为更严格的实测/半实物模型。
3. 本说明文件为重新生成版本。
